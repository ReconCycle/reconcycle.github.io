Developer guide
===============

The developer guide consits of manuals for developing new robotic skills, in-depth information on using robot control system, as well as manuals for adding new functionalities or (re-)training models of the vision system.

.. It also contains API documentation for the skill library and disassembly toolkit, which includes convenience functions for implementation of disassembly cycles.
